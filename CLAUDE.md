# rilog-lib ‚Äî CLAUDE.md

## What this is

Frontend monitoring library (npm: `@rilog-development/rilog-lib`). It intercepts HTTP requests (fetch + axios), click events, console errors/warnings, and custom log messages ‚Äî then batches them in localStorage and ships them encrypted to a backend (Rilog cloud, a local server, or a self-hosted endpoint).

### Data flow

1. `rilog.init({ key, config })` ‚Üí POST `/connection/init` ‚Üí receives `access_token`, `salt`, `recording` flag.
2. fetch/axios interceptors fire `onRequest` ‚Üí queue entry; `onResponse` ‚Üí dequeue + build `IRilogRequestItem`.
3. Each event goes through `pushEvents` ‚Üí stored in `localStorage['rilevents']`.
4. Flush triggers when: array > 10 events **and** queue is empty, localStorage nears capacity (4.5 MB), events exceed 2 MB, or long-poll timer fires (12 s inactivity).
5. Events are sorted by timestamp, AES-encrypted with salt, sent via `saveEvents`.
6. On `result === 'success'` the localStorage key is cleared.

### Request lifecycle & timeout

- Requests enter `QueueArray` on `onRequest`.
- A per-request `setTimeout(300 000 ms)` auto-evicts stale entries and records a timeout event.
- `dequeueNotResolved(olderThanMs)` is used during flush to capture in-flight requests older than `LONG_TIMER_LIMIT`.

### Send priority

`localServer` > `selfServer` > Rilog cloud (`saveEventsToRilog`).

## Public API (consumer-facing)

```ts
import rilog from '@rilog-development/rilog-lib';

rilog.init({ key: 'APP_KEY', config: { ... } });

// Axios (add to axios interceptors):
rilog.interceptRequestAxios(config);
rilog.interceptResponseAxios(response);

// Custom log:
rilog.logData(payload, { label: 'checkout', type: 'info' });
```

### TRilogInitConfig options

| Field | Purpose |
|---|---|
| `ignoredRequests` | URLs to skip entirely |
| `sensetiveRequsts` | URLs where headers+body ‚Üí `"sensetive"` |
| `sensetiveDataRequests` | URLs where only body ‚Üí `"sensetive"` |
| `headers` | Allowlist of headers to capture |
| `localStorage` | Allowlist of localStorage keys to capture |
| `disableFetchInterceptor` | Skip `window.fetch` patching |
| `disableClickInterceptor` | Skip click listener |
| `disableConsoleInterceptor` | Skip console wrap |
| `localServer` | `{ appName, params? }` ‚Äî rilog-local-logger |
| `selfServer` | `{ url, headers? }` ‚Äî custom backend |
| `onPushEvent` | Callback per event |
| `onSaveEvents` | Callback on flush |
| `meta` | `{ environment, branch, framework, platform }` |

## Code style

- **TypeScript strict mode**, target `es5`, `commonjs`.
- **Prettier**: `printWidth: 200`, `tabWidth: 4`, single quotes, trailing commas.
- **tslint** (`tslint.json`) enforced before publish (`prepublishOnly`).
- Classes + interfaces for all core logic. Adapters implement interfaces (`IAxiosAdapter`, `IFetchAdapter`).
- `@logMethods` decorator on public methods of `Rilog` for dev-only console debug output.
- Typos in existing identifiers (`sensetive`, `libruaryRequest`, `queque`) are intentional ‚Äî do not rename them; they appear in types, constants, and method names throughout.
- Interfaces prefixed `I`, types prefixed `T` or `E` (enums).
- No external runtime dependencies except `crypto-js`.

## Build & release

```bash
yarn build          # generate-secrets.js ‚Üí tsc ‚Üí dist/
yarn test           # jest
yarn format         # prettier
yarn patch          # npm version patch ‚Üí format + changelogen + git add
yarn release        # npm publish --access public
```

- `changelogen` auto-generates `CHANGELOG.md` on version bump.
- Secrets (`BASE_URL`, salts) are injected by `scripts/generate-secrets.js` from `.env` and written to `src/secrets.ts` (gitignored).

## Key constraints

- Library targets browsers only (uses `window.fetch`, `localStorage`, `document`).
- `initFetchInterception` patches `window.fetch` globally ‚Äî only call once.
- Events enum `ERilogEvent` must stay in sync with the backend.
- `EVENTS_ARRAY_LIMIT` (10) flush only triggers when `requestsQueue.isEmpty()` ‚Äî do not flush mid-flight.
- Max localStorage budget: 4.5 MB total, 2 MB for events buffer.
