# Security Policy

## Supported Versions

Only the latest published version on npm receives security fixes.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older   | ❌        |

## Reporting a Vulnerability

**Please do not report security issues through public GitHub issues.**

Send a description of the issue to **rilog.ceo@gmail.com**. Include:

- A description of the issue and its potential impact
- Steps to reproduce or a minimal proof of concept
- Affected versions

You will receive a response within 72 hours. If the issue is confirmed, a fix will be released as soon as possible and credited to you in the changelog (unless you prefer to remain anonymous).
